# VSCode keybindings
| Keybinding |	Action |
| ------------- | -------------
| Cmd + b | Open side file explorer |
| Cmd + p | Go to file |
| Option + Enter | Select all occurrences of Find match|
| Cmd + . | Quickfix |
| Cmd + \ | Split Editor |
| Cmd + 2 | Go to Pane 2 |
| Ctrl + 2 | Go to Tab 2 |
| Ctrl + 2 | Go to Tab 2 |
| Ctrl + n | Create New File |
| Cmd + k| Go to Definition |
| Ctrl + r | Go to References |
| Cmd + g + c | Git Commit |
| Cmd + g + p | Git Push |
| Cmd + g + P | Git Pull |
| Cmd + t | Open Terminal |
